Introduction to MERN Stack

MEAN (MongoDB, Express.js, AngularJS (or Angular), and Node.js)[1] is a source-available JavaScript software stack for building dynamic web sites and web applications.[2] A variation known as MERN

replaces Angular with React.js front-end,[3][4] and another named MEVN use Vue.js as front-end. Because all components of the MEAN stack support programs that are written in JavaScript, MEAN

applications can be written in one language for both server-side and client-side execution environments. Though often compared directly to other popular web development stacks such as the LAMP stack,

the components of the MEAN stack are higher-level including a web application presentation layer and not including an operating system layer.[5]

Technologies Used

  1. MongoDB
  2. Express.js
  3. React
  4. Node.js
